### Research (Part 1)

**Tenant white-label theming in Tailwind v4:** Learned that `bg-[var(--accent)]` works natively in Tailwind v4 arbitrary value syntax — it generates `background-color: var(--accent);` at build time. The right approach for multi-tenant CSS vars is inline `style={{ "--accent": color }}` on a root element, then `bg-[var(--accent)]/10 text-[var(--accent)]` for active states. Using a separate `--accent` variable (not overriding `--primary`) keeps CT Website Co. branding intact while still white-labeling nav active states per-tenant.

**CSS custom property key indexing in TypeScript:** `CSSProperties` from React doesn't accept `--` prefixed keys (CSS custom properties). Must use `Record<string, string>` or a type assertion when setting CSS vars like `--accent`.
